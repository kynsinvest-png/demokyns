🔒 Politique de confidentialité
Collecte des données
L'application collecte uniquement les informations nécessaires à la gestion de votre mutuelle : nom, téléphone, cotisations, assistances.

Stockage
Les données sont stockées dans votre propre Google Sheets, sous votre contrôle exclusif. KynsDev n'a aucun accès à vos données.

Partage
Aucune donnée n'est partagée avec des tiers. Les SMS sont envoyés via HSMS.ci en utilisant vos propres identifiants.

Sécurité
Les mots de passe sont hashés (SHA-256). Les communications sont chiffrées (HTTPS). Un token secret protège l'accès à votre base de données.

Droits
Vous disposez d'un droit d'accès, de modification et de suppression de vos données via l'interface administrateur.
